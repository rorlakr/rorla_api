RORLa-API (로라 API)
========

: ROR Lab. 웹사이트 리뉴얼 공개 프로젝트

ROR Lab.이 시작된지 언 2년이 지났습니다. 그동안 사용하던, 그렇지만 그렇게 활성화 되지 못했던 공식 웹사이트를 리뉴얼을 하고자 합니다.

ROR Lab.의 공유정신은 웹사이트 리뉴얼에도 그대로 이어져 누구나 참여할 수 있는 공개 프로젝트로 진행하고자 합니다.
